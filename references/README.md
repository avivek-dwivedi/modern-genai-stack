# References

Curated, deduplicated index of all papers, reports, and blog posts referenced across the hub.

| File | What's in it |
|---|---|
| [papers.md](papers.md) | Full paper index organized by section |

## How to Use

When adding content to the hub, link papers from this file rather than embedding URLs inline. This keeps references **findable, deduplicated, and easy to update**.

### Example

Instead of writing:
```markdown
LoRA: https://arxiv.org/abs/2106.09685
```

Write:
```markdown
LoRA — see [references/papers.md](papers.md)
```

Or even better, embed the link directly:
```markdown
[LoRA](references/papers.md) — adds low-rank matrices to frozen weights
```
